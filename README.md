# image-editor-clone

Next.js (App Router) + React + TypeScript + Tailwind CSS UI for an AI image editor landing page + mock editor.

## Development

```bash
npm install
npm run dev
```

## 环境变量

在项目根目录的 `.env.local` 里设置：

- `OPENROUTER_API_KEY`：你的 OpenRouter API Key（不要提交到仓库）

## Commands

- `npm run lint`
- `npm run build`
- `npm run start`

## Notes

- The in-page "editor" is currently frontend-only (no generation backend wired up).
